# Banquet Images Directory

Place your banquet hall images here in this directory.

## How to Add Images:

1. Copy your banquet images to this folder
2. The image path should be: `/images/banquets/your-image-name.jpg`

## Example Usage in MongoDB:

```json
{
  "name": "Grand Ballroom",
  "capacity": 500,
  "price": 50000,
  "image": "/images/banquets/taj-palace-banquet.jpg"
}
```

## Supported Formats:
- JPG / JPEG
- PNG
- WebP
- GIF

## Image Recommendations:
- Width: 800px or more
- Height: 600px or more
- File size: Less than 2MB for optimal loading
